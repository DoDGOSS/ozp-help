HUD (Heads Up Display)

 

Overview

HUD is a repository for all listings that you bookmark in Marketplace. From HUD, you can quickly find, launch or get help relating to each listing. Also, you can remove the bookmarked listing.

Accessing HUD

You can access HUD two ways using the Global Toolbar:

  • Click the icon on the upper-left side of the toolbar or click the  icon on the far-right side of the toolbar, then select Bookmarks:
    Accessing HUD from the Global Toolbar

 

Listing Actions from HUD

From HUD you can open, get help or remove bookmarks for a listing. Each action is found under the drop-down listing menu:

Drop-down listing actions menu

This section describes listing commands--accessible from the drop-down arrow on the right side of the listing:

  • Launch in Webtop -- Opens the listing in your HUD.
  • Launch in new tab -- Opens the listing in a new browser tab.
  • Get help -- Opens the listing's Help resources.
  • Remove Bookmark -- Deletes the listing from your HUD. To add it again, find and bookmark the listing in Marketplace.

 

Folders in HUD

Creating Folders

Like a smart phone, the HUD supports folders.

To make a folder:

  1. Click a listing and drag it over another listing. Release the listing to create a folder.
  2. Drag more listings into the folder to add them to it.

Naming Folders

As shown in the image above, your first folder defaults to the name "New Folder." To change a folder name:

  1. Double-click the folder name which highlights the textbox.
  2. Type the new name in the textbox.
  3. Click anywhere else on the screen. This will save the new folder name.

Remove Listings from a Folder

To remove a listing from a folder:

  1. Click once on the folder to open it.
  2. Drag the listing from the folder. Expect it to disappear.
  3. When you close the folder (click the X), you will see the listing in your HUD--separate from the folder as shown by the MusicBox listing in the image above.

Edit or Delete a Dashboard

After creating a dashboard you can change its name or view. However, if you change its view, you will lose the state of the listings on that dashboard. Therefore when the dashboard resets you will lose any data loaded in that dashboard.

To edit a dashboard:

  1. Click at the bottom-left side of Webtop's lower toolbar.
    Note: Your first dashboard is called Default. If you change that name or use a different dashboard, you will see a different name to the left of the arrow.
  2. Click the edit icon beside the dashboard that you want to change:
  3. The edit window opens. Change the Name or View.
    Note: If you change the view, you will lose the state of the listings on the dashboard.
  4. Click Save Dashboard.

Delete a Dashboard

To delete a dashboard:

  1. Click at the bottom-left side of Webtop's lower toolbar.
  2. Click the trash can icon beside the dashboard that you want to delete.
  3. A warning appears, click OK to confirm that you want to delete the dashboard.
  4. The system will remove the dashboard. You cannot undo this action.

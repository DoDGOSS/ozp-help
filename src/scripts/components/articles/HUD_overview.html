HUD Overview

HUD is a repository for all listings that you bookmark in Center. From HUD, you can quickly find, launch or get help relating to each listing. Also, you can remove bookmarked listing.

Listing Actions from HUD

From HUD you can open, get help or remove bookmarks for a listing. Each action is found under the drop-down menu each listing has in its upper-right corner:

Drop-down listing actions menu

Listing commands--accessible from the drop-down menu on the upper-right side of the listing:

  • Launch in Webtop -- Opens the listing in your HUD.
  • Launch in new tab -- Opens the listing in a new browser tab.
  • Get help -- Opens the listing's Help resources.
  • Remove Bookmark -- Deletes the listing from your HUD. To add it again: go to Center, search for the listing and bookmark it.

HUD Overview

HUD is a repository for all listings that you bookmark in Center. From HUD, you can quickly find, launch or get help relating to each listing. Also, you can remove bookmarked listing.

HUD for Stewards

Stewards have an additional section on their HUD for listing management:

All users have the My Bookmarks section of HUD, only listing owners see My Listings. However, only stewards see the listing management section described below:

From HUD, stewards see the number of listings that need action or are approved, pending, returned or in draft format in the system. To take action on any of the listings, click the Listing Management link. Also, the steward section includes Recent Activity about each listing you oversee in the system. From each entry, you can view more information using the See the listing changes link or open the listing by clicking on its name.
